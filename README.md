# Student Record Management System (C)

## 📌 Project Overview

Student Record Management System is a beginner-friendly console application developed using **C Programming**. The project demonstrates the fundamentals of structured programming by allowing users to manage student records through a menu-driven interface.

Users can add new student records, display all stored records, search students using their roll number, and exit the application. The project is designed to strengthen programming fundamentals such as structures, arrays, loops, functions, conditional statements, and user input handling.
